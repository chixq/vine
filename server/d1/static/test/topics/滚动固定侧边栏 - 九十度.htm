滚动固定侧边栏

使用 jQuery 保持滚动对象在窗口的可见范围内的。

例子：苹果在线商店的购物车

演示

方法

添加 CSS

.fixed {   
    position:fixed; /* IE 6 不支持 fixed，只能用 absolute */
    top:20px;   
    background-color:#90EE90 /* 可选 */  
}

添加 jQuery

  1. $(document).ready(function (){
  3.   // 检查对象，这里是 HTML 5 的 "aside"，可改为 ".class" 或者 "#id"
  4.   var offset = $("aside").offset();
  5.   $(window).scroll(function() {
  6.     var scrollTop = $(window).scrollTop();
  7.     // 视窗还看到这个侧边栏？
  8.     if(offset.top < scrollTop) {
  9.       // 没看到，添加 ".fixed"
  10.       $("aside").addClass("fixed")
  11.     }else {
  12.       // 看到，不需要 ".fixed"
  13.       $("aside").removeClass("fixed")
  14.     }
  15.   })
  17. });

注意：
1、演示基于 HTML5 模板，aside 相等于侧边栏
2、IE 1-8 不支持 HTML5 的标签，需要用到 Modernizr 或者 html5shiv（差别 2KB 左右）
3、如果不用 HTML5 的标签，不需要理会上面2条提示
